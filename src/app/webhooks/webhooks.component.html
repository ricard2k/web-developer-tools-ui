Webhooks

A webhook is a method of augmenting the behavior of an application with custom callbacks executed server to server. Webhooks are typically used to provide real-time data updates and notifications. When an event occurs in an external service, it triggers the webhook, which sends an HTTP request to the specified application with the event data.

Unlike redirections, webhooks operate asynchronously and are triggered by events, redirections are synchronous and occur as part of the user's navigation flow. Having out-of-band communication makes web applications more secure by decoupling the event processing from the user web flow flow. This reduces the risk of exposing sensitive data or logic to the client side and allows for more controlled and secure handling of events.
